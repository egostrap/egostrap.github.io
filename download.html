Installation Guide

EgoStrap is designed to be lightweight and simple to set up. It doesn't modify the Roblox launcher files permanently; instead, it registers as a custom bootstrapper handler.

Microsoft .NET Framework Requirement

EgoStrap requires the Microsoft .NET Desktop Runtime 6.0 or higher. Most Windows 10 & 11 PCs have this pre-installed, but if yours lacks it, the installer will guide you through downloading it.

Step-by-Step Installation

  1. Navigate to the Download Page and download EgoStrap-Setup.exe.
  2. Double-click the downloaded installer.
  3. Windows SmartScreen may trigger a warning because EgoStrap is a newer, open-source utility without an expensive digital signing certificate. Click More Info, then click Run Anyway.
  4. Select the installation directory (we recommend keeping the default: `%localappdata%\EgoStrap`).
  5. The program will run a brief installation procedure, configure the handler protocol, and load the dashboard interface.

Uninstalling EgoStrap

If you wish to restore the stock Roblox bootstrapper settings:

  1. Open the EgoStrap Config Dashboard.
  2. Go to the bottom of the General tab, and click Uninstall.
  3. EgoStrap will cleanly unregister its custom handlers, restore standard Roblox launcher defaults, and wipe its directory. No stray registry keys or cache folders are left behind.

First Launch Guide

Once installation finishes, you will be greeted by the EgoStrap Config Dashboard. This is your control hub for Roblox.

How EgoStrap Handles Roblox Starts

EgoStrap registers the custom URI handler protocol `roblox-player:`. Whenever you click **Play** on the Roblox website or open the Roblox Desktop shortcut, Windows directs the request to EgoStrap instead of the default Roblox bootstrapper.

What happens during boot?

  1. EgoStrap launches in the background.
  2. It scans for Roblox client updates on the official Roblox deployment servers.
  3. It injects custom mods, configurations, fonts, and client settings locally.
  4. It unlocks framerates by writing dynamic configurations to client settings.
  5. It launches the official Roblox Player process.

Do Not Launch as Administrator

Running EgoStrap with administrator privileges can prevent it from communicating with browser launch protocols. Run the application normally as a standard user.

Core Settings Guide

EgoStrap's General settings tab houses options for adjusting how the bootstrapper behaves during startup and interacts with external widgets.

Automatic Client Updates

Roblox pushes mandatory updates multiple times a week. With Auto-Updates toggled on, EgoStrap detects outdated builds during launch, downloads only the modified asset chunks, updates them locally, and runs the game in seconds.

Discord Rich Presence (DRP)

Let your friends see what you are playing! DRP displays the name of the place, your current character thumbnail, active session duration, and server status on your Discord profile automatically.

To configure DRP, open the General tab, toggle Discord Rich Presence, and restart the client. Make sure Discord is running and game status display is enabled in Discord user settings.

Custom UI Themes

EgoStrap allows you to customize the bootstrapper's appearance and elements of the Roblox game interface natively.

Modifying the Bootstrapper Theme

You can select color themes directly inside the Dashboard. Custom styles adjust borders, backdrop grids, typography, and active highlights. Supported themes include Dark, Light, Cyberpunk Pink, and Minimal Blue.

Sound Overrides

Roblox replaced the classic "Oof" sound with a modern sound in 2022. EgoStrap lets you restore the classic "Oof" sound natively. In the customization tab, select Classic Oof (Pre-2022). EgoStrap will override the sound asset locally during launch.

Mod Manager Studio

EgoStrap has a dedicated Mod Manager folder. Drop custom files in the folders, and EgoStrap mounts them over standard Roblox files dynamically when launching.

How to Install Custom Mods

  1. Open the EgoStrap Dashboard.
  2. Go to the Customization/Mod tab, and click Open Mods Folder.
  3. A folder window will open on your PC. It contains folders for:
    • `sounds/` — Drop custom wave/mp3 sounds.
    • `textures/` — Replace cursor styles, character overlays, etc.
    • `fonts/` — Install custom typefaces.
  4. Move your custom assets into the respective folder.
  5. Launch the game. EgoStrap loads your custom modifications automatically!

Account Profiles Manager

Play on multiple accounts? EgoStrap has a secure local profile manager that lets you switch accounts without entering your credentials repeatedly in a browser.

Security and Cookie Encryption

EgoStrap stores login session tokens locally on your PC inside a securely encrypted database. They are only ever shared directly with Roblox's authentication endpoints, never with any third-party servers.

Adding and Swapping Accounts

  1. Log into your Roblox account in your browser.
  2. Open the EgoStrap Dashboard, navigate to the Account Profiles tab.
  3. Click Import Active Browser Session. EgoStrap will automatically identify your active session token.
  4. Name the profile (e.g., "AltAccount01").
  5. To swap accounts, select the profile in EgoStrap and click Activate. The next time you hit play, Roblox will load with that account!

FastFlags configurations

FastFlags are configurations used by Roblox developers to control game engine features. EgoStrap gives you a user-friendly panel to modify these values directly.

Common FastFlags Settings

  • Unlock Framerate: Removes the 60fps limit and enables high framerates.
  • Render Distance Scale: Extends rendering capabilities, allowing you to see objects further away.
  • Shadow Mapping Precision: Toggles high-resolution shadows for cleaner graphics.

Importing Custom FastFlag Lists

If you have a custom JSON list of FastFlags, you can import it into EgoStrap:

{ "DFIntTaskSchedulerTargetFps": 9999, "FFlagDebugGraphicsDisableDirect3D10": "True", "FFlagDisablePostFx": "True" }

Go to the FastFlags console, click Import JSON, paste your configurations, and click Save.

Vulkan & D3D Graphics Tuning

Roblox auto-selects your graphics engine API, which doesn't always yield the best performance on all graphics cards. EgoStrap lets you override the backend rendering engine.

API Profiles

  1. Vulkan: Excellent performance on AMD graphics cards and modern PCs. Provides higher frame rates but may cause stuttering on older Nvidia GPUs.
  2. Direct3D 11: The default rendering engine. Very stable, good for standard hardware.
  3. Direct3D 12: Best for modern Windows 10/11 PCs with dedicated GPUs, supporting advanced shader pipelines.

Modify these settings in the FastFlags Studio tab inside the Config Dashboard.

Troubleshooting Guide

Encountering issues with EgoStrap? Here are solutions to common problems.

Roblox Launches with stock settings (No FPS Unlock / Mods)

This typically occurs if the file handler association is broken. To fix this:

  1. Open the EgoStrap Dashboard.
  2. Click the Repair Handler Links button in the bottom-left of the Dashboard.
  3. Select Yes when prompted for handler repair.

Game Crashes on Launch

If your game crashes immediately during startup, it is likely caused by an invalid or corrupted FastFlag configuration.

  1. Open the Dashboard and go to the FastFlags Studio tab.
  2. Click Reset to Defaults to clear all custom flag overrides.
  3. Attempt to launch the game again.
