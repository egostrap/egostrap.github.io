Frequently Asked Questions

Got a question? We have answers. If you can't find what you need here, join our official Discord server.

Can I get banned for using EgoStrap?

No, you cannot be banned for using EgoStrap. Roblox's anti-cheat software (Hyperion) targets software that injects code, reads/writes runtime memory (such as exploits or cheat modules), or executes arbitrary scripts inside Roblox servers.


EgoStrap is client-side only. It works side-by-side with Roblox by modifying startup arguments, configuring FastFlags (settings native to the game engine), and overriding local files in your system directories. It does not touch active memory or behave like an exploit.

Does EgoStrap cost money?

No, EgoStrap is entirely free and open-source. We do not lock features behind paywalls, run advertising scripts, or charge subscriptions. If you would like to support development servers and domain costs, you can make a small contribution on our Donate page.

How do I unlock my FPS with EgoStrap?

Unlocking FPS is enabled by default when you install EgoStrap. You can adjust your framerate preferences inside the EgoStrap Config Dashboard by navigating to the FastFlags Studio tab, toggling the Framerate Unlocker setting, and setting your target value (e.g., 144, 240, or 9999 for unlimited).

How do I import custom sounds and textures?

It is simple! Open the EgoStrap Dashboard, navigate to the Customization tab, and click Open Mods Folder. Drop your audio wave files into the `sounds` directory, or override image panels in the `textures` folder. The bootstrapper handles mapping them to Roblox at startup automatically. See our Mods Documentation for details.

Does EgoStrap run on macOS or Mobile devices?

Currently, EgoStrap only supports Windows (Windows 10 and 11, 64-bit systems). macOS and mobile operating systems utilize different launcher file distribution standards that do not support custom bootstrapper handler redirection.

Why does my antivirus flag EgoStrap?

Because EgoStrap edits client registry parameters, creates folder hooks, and launches background threads to check for updates, some aggressive antivirus software (such as Windows Defender SmartScreen) may flag the installation.


EgoStrap is fully open-source. You can inspect every line of code on our official GitHub repository to verify its safety. If flagged, you can safely select "Run Anyway" or add EgoStrap to your antivirus exemption list.
