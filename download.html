Quick Installation Guide

  1. Download the EgoStrap-Setup.exe package.
  2. Run the installer. If prompted by Windows Defender SmartScreen, select More Info and click Run Anyway (occurs because the installer is fresh and unsigned).
  3. The EgoStrap Dashboard will open automatically. Customize your settings, toggle FPS limits, and click Launch Roblox!
